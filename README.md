# BreadFeet

경북대학교 2025 2학기 종합설계프로젝트1 
지도 &amp; 위치 서비스 웹 개발 프로젝트

## 주요기능
- 지도 위치

## 기술스택
- Frontend: React
- 스타일링: CSS
- 지도 API: Kakao Maps
- 기타: Axios, React Router

## 파일구성
src  
  |- assets/: 이미지, 아이콘, 전역 스타일 등  
  |- components/: 공용 컴포넌트  
  |- pages/: 페이지 단위 컴포넌트  
  |- ...  
  |- App.jsx  
  |- App.css  
  |- main.jsx  
  |_ index.css
